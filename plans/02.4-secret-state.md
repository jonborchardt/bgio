# 02.4 — Secret state via playerView

**Parent:** [02-engine.md](02-engine.md)

## Goal

Each role only sees its own hidden information. The Foreign player should
not see the deck order or science card requirements while in flight; the
Domestic hand is private to whoever holds Domestic; etc.

## Files

- Create `src/game/playerView.ts`.
- Modify `src/game/index.ts` to set `playerView`.
- Create `tests/playerView.test.ts`.

## API

```ts
import type { PlayerID } from './types';

// bgio's playerView signature: (G, ctx, playerID) => PartialG
export const playerView: (
  G: SettlementState,
  ctx: Ctx,
  playerID: PlayerID | null,
) => SettlementState;

// Internal helpers used by the redactor.
export const redactDeckOrder = <T>(deck: T[]) => Array(deck.length).fill(null) as T[];
export const redactHand = <T>(hand: T[]) => Array(hand.length).fill(null) as T[];
```

## Steps

1. Build a per-role redaction table:
   - `chief` is fully public (chief actions are visible to everyone).
   - `science`: tech cards under each science card are face-up (everyone
     sees them); the per-card paid resources are public; nothing private.
   - `domestic`: hand is private to whoever holds Domestic.
   - `foreign`: hand, battle-deck order, trade-deck order, and the in-flight
     battle's drawn-but-not-yet-resolved card are private to whoever holds
     Foreign.
2. `playerView` resolves which roles `playerID` holds via
   `rolesAtSeat(G.roleAssignments, playerID)`. Anything not held by that
   player gets redacted.
3. `playerID === null` → spectator view: redact everything that any role
   would redact (treat as "viewer holds zero roles").
4. Decks redact via `redactDeckOrder`; hands via `redactHand`. Counts stay
   visible (replace cards with `null`, keep array length).

## Tests

- Player 1 in a 2-player game (Domestic+Foreign): sees own Domestic hand,
  cannot see the Foreign hand of player 0 (who is Chief+Science) — but in
  this assignment player 1 *is* Foreign, so they see it. Construct cases
  for both directions.
- Spectator view (`playerID = null`) redacts every Foreign deck and
  Domestic hand.
- Visible counts match real array lengths.
