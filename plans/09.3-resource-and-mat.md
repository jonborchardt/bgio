# 09.3 — ResourceBag & CenterMat components

**Parent:** [09-ui.md](09-ui.md)

## Goal

A `<ResourceBag>` component for displaying any `ResourceBag`, and a
`<CenterMat>` showing every per-player circle and the trade-request slot.

## Files

- Create `src/ui/resources/ResourceBag.tsx`.
- Create `src/ui/resources/ResourceChip.tsx` — single resource, count.
- Create `src/ui/mat/CenterMat.tsx`.
- Create `src/ui/mat/Circle.tsx` — single circle (per-player).
- Create `src/ui/mat/TradeRequestSlot.tsx`.
- Create `tests/ui/resources/ResourceBag.test.tsx`,
  `tests/ui/mat/CenterMat.test.tsx`.

## API

```tsx
export interface ResourceBagProps {
  bag: ResourceBag;
  hideZero?: boolean;          // default true
  size?: 'sm' | 'md' | 'lg';
}
export const ResourceBag: React.FC<ResourceBagProps>;

export const ResourceChip: React.FC<{ resource: Resource; count: number; size?: 'sm'|'md'|'lg' }>;

export const CenterMat: React.FC<BoardProps<SettlementState>>;
```

## Steps

1. `ResourceChip`: MUI `Chip` whose color comes from
   `palette.resource[resource]` (added in 09.4). Icon → small SVG / emoji
   keyed by resource name.
2. `ResourceBag` renders chips for resources with `count > 0` (configurable
   by `hideZero`).
3. `CenterMat`:
   - For each non-chief seat: render `<Circle seat>` showing the bag.
   - Render `<TradeRequestSlot>` whether `mat.tradeRequest` is null or set.
   - Click a circle (and you hold its seat) → fires `pullFromMat(amount)`
     via context-injected moves.
4. Layout: a card-table look — circles arranged around the trade slot.

## Tests

- `ResourceBag` shows only non-zero resources by default.
- `ResourceBag hideZero={false}` shows everything.
- `CenterMat`: clicking your own circle fires `pullFromMat` with the
  current circle bag.
- Trade slot renders nothing when `tradeRequest` is null.

## Dependencies
