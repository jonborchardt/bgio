## Steps

1. **Flip battle:**
   - Validate caller seat holds `foreign` and is in `foreignTurn`.
   - Validate `inFlight.battle === null`.
   - Pop top of `battleDeck` into `inFlight.battle`. Set `inFlight.committed
     = G.foreign.inPlay` (initial commit; player can re-commit before
     assigning damage in V2 â€” V1 simplification).
   - Restore HP of all in-play units (game-design.md "Restore the health of
     all units before each flip").
   - Set seat's stage to `foreignAwaitingDamage`.

2. **Assign damage:**
   - Validate stage `foreignAwaitingDamage`.
   - Run `resolveBattle(input)` from 07.3.
   - If `outcome === 'mid'` or validationErrors â†’ `INVALID_MOVE`.
   - On `'win'`:
     - Apply card's `reward` to bank.
     - Increment `G.settlementsJoined` if card flagged `joins`.
     - Move stage to `foreignTurn` (player may flip more or end).
   - On `'lose'`:
     - Schedule tribute for next round (`G.foreign.pendingTribute = card.failure.tribute`)
       â€” applied in chief's next phase.
     - End the foreign player's stage (set `_phaseDone[seat]=true` semantics
       per 02.2).
   - Clear `inFlight.battle = null`.

3. **Flip trade:**
   - Validate stage `foreignTurn`, last battle outcome was `'win'`.
   - Pop top of `tradeDeck` into a new `TradeRequest`.
   - Validate mat has no existing trade request OR move into the
     `awaitingChiefDecision` stage so chief picks which to keep.
   - On chief decision: `mat.tradeRequest = chosen`.
