# 05.3 — scienceComplete move

**Parent:** [05-science.md](05-science.md)

## Goal

When a card's `paid` ledger covers its `cost`, complete it: distribute its
4 underlying tech cards to the right roles per game-design.md §Science
("Red to Foreign, Gold to Chief, Green to Domestic, Blue to Science").

## Files

- Create `src/game/roles/science/complete.ts`.
- Register move in `src/game/moves.ts`.
- Create `tests/roles/science/complete.test.ts`.

## API

```ts
export const scienceComplete: Move<SettlementState>;
// args: cardID: string
```

## Steps

1. Validate caller is science seat in stage `scienceTurn`.
2. Validate card not yet completed.
3. Validate `paid[cardID]` covers `cost` (i.e. `canAfford(paid, cost)`).
4. Validate `G.science.perRoundCompletions < 1` (rule: "only 1 a turn").
5. Mark `completed.push(cardID)`. Increment counter.
6. Move `paid[cardID]` to bank (resources have been spent — they go to the
   bank for accounting).
7. Distribute the 4 tech cards in `underCards[cardID]`:
   - red → seat holding `foreign` → `G.foreign.hand`
   - gold → seat holding `chief` → `G.chief.hand`
   - green → seat holding `domestic` → `G.domestic.hand`
   - blue → seat holding `science` → `G.science.hand`
8. If a card's color isn't in {red, gold, green, blue} (data error), throw
   loudly so the test catches it.

## Tests

- Successful completion moves the card to `completed`, resources to bank,
  and distributes 4 tech cards to the four target hands.
- Trying to complete with `paid < cost` → `INVALID_MOVE`.
- Trying to complete twice in one round → second call returns
  `INVALID_MOVE`.
- After `endOfRound`, the per-round counter resets and a new completion is
  allowed.
- Distribution honors role assignments in 1p / 2p / 3p / 4p configurations
  (in 1p, all four hands are on seat 0).

## Dependencies

01.1, 02.5 (round-end resets counter), 03.2 (bank), 05.1, 05.2.

## Out of scope

- Tech-card *effects* on hand owners (consumed by per-role moves).
- The "may add resources to any number of cards but may complete only 1 a
  turn" — that's exactly the per-round counter; nothing else to do here.
