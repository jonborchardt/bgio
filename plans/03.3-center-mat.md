# 03.3 — Center mat: per-player circles & trade-request slot

**Parent:** [03-resources.md](03-resources.md)

## Goal

Model the central play area: one resource circle per non-chief player, plus
a single "trade request" slot the Foreign player can place into.

## Files

- Modify `src/game/types.ts`: add `centerMat: CenterMat`.
- Create `src/game/resources/centerMat.ts`.
- Create `tests/resources/centerMat.test.ts`.

## API

```ts
import type { ResourceBag, PlayerID } from '../types';

export interface CenterMat {
  circles: Record<PlayerID, ResourceBag>; // one per non-chief seat
  tradeRequest: TradeRequest | null;       // 0 or 1 active
}
export interface TradeRequest {
  id: string;
  ownerSeat: PlayerID;     // the seat acting as the trading partner (mat circle)
  required: ResourceBag;
  reward: ResourceBag;
}

export const initialMat = (assignments: Record<PlayerID, Role[]>): CenterMat;
export const placeIntoCircle = (mat: CenterMat, seat: PlayerID, amounts: Partial<ResourceBag>) => void;
export const pullFromCircle = (mat: CenterMat, seat: PlayerID, amounts: Partial<ResourceBag>) => void;
export const setTradeRequest = (mat: CenterMat, req: TradeRequest) => void;
export const clearTradeRequest = (mat: CenterMat) => void;
```

## Steps

1. `initialMat` builds a circle per non-chief seat (chief seat = no circle).
   Single-player game (all roles on seat 0): no circles, no mat motion
   needed; setup degenerates gracefully.
2. `placeIntoCircle` / `pullFromCircle` are Immer-mutating; both throw on
   missing seat or underflow. Pulling more than available throws.
3. `setTradeRequest` throws if a request already sits there — caller
   (Foreign flow / Chief discard helper) is responsible for clearing first.
4. **Round-end hook:** in this sub-plan, register a hook with 02.5 named
   `mat:sweep-leftovers` that:
   - moves leftover circle contents to the bank,
   - leaves any trade request alone (those persist across rounds until
     fulfilled or discarded).

## Tests

- Setup: 4-player game produces 3 circles (one per non-chief seat).
- 1-player game: zero circles; mat helpers no-op without throwing.
- Round-end hook empties circles into bank; total resources conserved.
- Pulling more than circle contents throws.
- `setTradeRequest` rejects placing on a non-empty slot.

## Dependencies

03.1, 03.2, 02.5 (hook registry), 01.1 (role assignments).

## Out of scope

- The Foreign-side "what to do when Chief must choose which trade request to
  discard" flow — that's 04.x or 07.5.
