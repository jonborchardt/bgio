# 03.2 — Bank slice & transfer primitives

**Parent:** [03-resources.md](03-resources.md)

## Goal

A single bank slice on `G` plus tested transfer primitives. The bank is the
"source of truth" for resource conservation across the game.

## Files

- Modify `src/game/types.ts`: refine the `bank: ResourceBag` field typing.
- Create `src/game/resources/bank.ts`.
- Create `tests/resources/bank.test.ts`.

## API

```ts
import type { Resource, ResourceBag } from './types';

// Pure transfer between any two bags, used internally by all moves that
// move resources around. Mutates Immer drafts.
export const transfer = (
  from: ResourceBag, to: ResourceBag, amounts: Partial<ResourceBag>,
) => void;

// Initial bank. Default per game-design.md §Setup.Chief: 3 gold.
// Tunable per match via setupData (lobby form passes a `startingBank`
// override). Fall back to default when unset.
export const initialBank = (override?: Partial<ResourceBag>): ResourceBag;

// Asserts conservation across a state transition. For tests + dev-mode.
export const totalResources = (G: SettlementState): number;
```

## Steps

1. `transfer` first checks affordability via `canAfford`; throws `RangeError`
   on insufficient `from` (we want loud failure inside moves so the caller
   can convert to `INVALID_MOVE`).
2. `initialBank` returns `bagOf({ gold: 3 })` by default.
   `initialBank({ gold: 5 })` returns `bagOf({ gold: 5 })`. The lobby's
   match-create form (10.3) passes a `setupData.startingBank` override
   that flows into `setup`.
3. `totalResources` walks every resource holder on `G` (bank, circles,
   trade-request slot, per-player wallets if any) and sums them. Callers in
   dev-mode can assert pre/post equality.

## Steps (state shape)

- `G.bank` is `ResourceBag`, present from setup.
- Setup function uses `initialBank()`.

## Tests

- `transfer({gold:5,...}, EMPTY, {gold:3})` results in `{gold:2,...}` and
  `{gold:3,...}`.
- `transfer` underflow throws.
- `totalResources` is invariant across any sequence of `transfer` calls
  between holders on `G`.
- `initialBank()` returns 3 gold; `initialBank({ gold: 5 })` returns 5;
  `initialBank({ wood: 2 })` returns `{ gold: 0, wood: 2, ... }` (override
  fully replaces, doesn't merge with default).
