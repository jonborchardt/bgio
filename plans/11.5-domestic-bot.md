# 11.5 — DomesticBot

**Parent:** [11-ai.md](11-ai.md)

## Goal

Heuristic for Domestic: buy any affordable building, upgrade if no buys are
affordable, prefer cells that activate adjacency bonuses (06.5).

## Files

- Create `src/game/ai/domesticBot.ts`.
- Create `tests/ai/domesticBot.test.ts`.

## API

```ts
export const domesticBot: { play: (state) => BotAction };
```

## Steps

1. Sort `hand` by `cost` ascending.
2. For each hand card affordable from wallet:
   - For each legal placement cell:
     - Compute hypothetical adjacency bonus via 06.5's pure helper.
   - Pick the highest-bonus cell (ties → topmost-leftmost).
3. If a cell+card found: `domesticBuyBuilding(card, x, y)`.
4. Else: try cheapest affordable upgrade.
5. Else: skip to `domesticProduce`.
6. After `produce`: end phase.

## Tests

- With one affordable card and a placement that triggers a bonus, bot
  picks that cell.
- With nothing affordable, bot calls `produce` only.
- After `produce`, bot does not call it again that round.

## Dependencies

06, 11.2.

## Out of scope

- Long-horizon planning (which upgrade chain to commit to).
