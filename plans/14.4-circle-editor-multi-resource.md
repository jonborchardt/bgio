# 14.4 — Multi-resource CircleEditor

**Parent:** [14-playtest-followups.md](14-playtest-followups.md)

## Goal

`chiefDistribute` accepts any `Partial<ResourceBag>` (gold, wood,
stone, steel, horse, food, production, science, happiness, worker).
Today the UI hardcodes a `+1 / +2 / +5` gold stepper only — see
[../notes/playtest-2026-04-29.md](../notes/playtest-2026-04-29.md)
issue V4. Once Domestic produce yields non-gold resources (06.4)
and Science contribute consumes them, the chief needs to be able
to redistribute them.

## Files

- Modify `src/ui/chief/CircleEditor.tsx` — render a row per
  non-zero bank resource, not just gold.
- Modify `src/ui/chief/ChiefPanel.tsx` — pass the full bank bag
  through; collapse rows for resources the bank doesn't hold.

## API

```tsx
export interface CircleEditorProps {
  seat: PlayerID;
  circle: ResourceBag;
  bank: ResourceBag;
  canPush: boolean;
  onPush: (resource: Resource, amount: number) => void;
}
```

(Same shape as today; the change is internal — render every
non-zero resource in `bank` instead of the hardcoded gold row.)

## Steps

1. Iterate `RESOURCES`; skip any with `bank[r] === 0`.
2. For each surviving resource, render a row with the resource icon
   (theme palette accent), current circle amount, and three buttons
   `+1 / +2 / +5`. Disable a button when `bank[r] < amount`.
3. `onPush(resource, amount)` calls
   `props.moves.chiefDistribute(seat, { [resource]: amount })`.
4. Visual: Stack rows vertically; the chief sees one row per
   resource the bank actually has. The visual height grows but the
   chief panel scrolls within its own container.

## Tests

- Bank `{ gold: 3, wood: 2 }` renders two rows.
- Bank `{ gold: 0, wood: 0 }` renders zero rows (nothing to
  distribute).
- Clicking `+2 wood` calls `chiefDistribute(seat, { wood: 2 })`.
- `+5` is disabled when bank has 3.

## Dependencies

03.1 (RESOURCES), 03.2 (initialBank), 04.1 (chiefDistribute), 04.5
(ChiefPanel), 09.4 (theme tokens for per-resource accent).

## Out of scope

- Bulk-distribute mode (send N gold to all seats with one click).
- Drag-drop interaction; clicks are enough for V1.
