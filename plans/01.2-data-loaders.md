# 01.2 — Typed JSON data loaders

**Parent:** [01-foundations.md](01-foundations.md)

## Goal

Wrap [src/data/*.json](../src/data/) in typed loaders that validate at module
load time. Logic everywhere else imports from `src/data/index.ts`, never
directly from raw JSON.

## Files

- Create `src/data/index.ts` — barrel + typed exports.
- Create `src/data/schema.ts` — interfaces + a tiny ad-hoc validator (no
  external dep yet; if we add `zod` later it goes here).
- Create `tests/data.test.ts`.

## API

```ts
// src/data/schema.ts
export interface BuildingDef {
  name: string; cost: number; benefit: string; note: string;
}
export interface UnitDef {
  name: string; cost: number; initiative: number;
  attack: number; defense: number;
  altStats: string; requires: string; note: string;
}
export interface TechnologyDef {
  branch: string; name: string; order: string; cost: string;
  buildings: string; units: string;
  blueEvent: string; greenEvent: string; redEvent: string; goldEvent: string;
}

export const validateBuildings = (raw: unknown): BuildingDef[];
export const validateUnits = (raw: unknown): UnitDef[];
export const validateTechnologies = (raw: unknown): TechnologyDef[];

// src/data/index.ts
export const BUILDINGS: ReadonlyArray<BuildingDef>;
export const UNITS: ReadonlyArray<UnitDef>;
export const TECHNOLOGIES: ReadonlyArray<TechnologyDef>;
```

## Steps

1. `import buildingsRaw from './buildings.json'` — Vite typed-JSON works
   with `resolveJsonModule` already enabled in tsconfig.
2. Each `validate*` function: type-narrows by checking required fields and
   their primitive types; throws with the offending index on failure.
3. `index.ts` runs the validators at module load and exports frozen
   `ReadonlyArray`s. Failures crash early (intentional).
4. Add a `BENEFIT_TOKENS` constant listing the verbs we know how to parse
   (`food`, `production`, `science`, `gold`, `attack`, `defense`,
   `happiness`, `unit maintenance`) — referenced by 06.3.

## Tests

- All three loaders produce non-empty arrays.
- Mutating `BUILDINGS[0].cost = 0` throws (frozen).
- A hand-rolled malformed input (e.g. `[{ name: 1 }]`) throws via the
  validator with a clear message.

## Dependencies

None — runs against the JSON already in `src/data/`.

## Out of scope

- Translating `benefit` strings to typed yields (06.3).
- Tech card filtering by branch / level (05.1).
