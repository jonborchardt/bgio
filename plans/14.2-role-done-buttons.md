## Goal

Provide a real (non-test-only) move for each non-chief role to flip
`G.othersDone[seat] = true`, plus a footer button on each role panel
that fires it. Without this, `othersPhase`'s `endIf` never trips
once the test scaffolds are stripped (review fix #1 just landed
that strip), so a real session deadlocks after the chief ends their
phase. Also remove the legacy bottom "End my turn" stub from
`Board.tsx` â€” see [../notes/playtest-2026-04-29.md](../notes/playtest-2026-04-29.md)
issue V1 / E1 / E3.

## API

```ts
export const scienceSeatDone:  Move<SettlementState>; // no args
export const domesticSeatDone: Move<SettlementState>; // no args
export const foreignSeatDone:  Move<SettlementState>; // no args
```

## Steps

1. Each move:
   - validates `playerID !== undefined`,
   - validates `rolesAtSeat(roleAssignments, playerID).includes('<role>')`,
   - validates `ctx.activePlayers?.[playerID] === '<role>Turn'`
     (or `'foreignAwaitingDamage'` for foreign â€” both should mark
     the seat done),
   - sets `G.othersDone[playerID] = true`.
2. The bgio engine re-checks `othersPhase.endIf` after each move,
   so once every non-chief seat has flipped done, bgio transitions
   to `endOfRound` automatically.
3. Each role panel renders a footer "End my turn" button gated on
   the active stage. Clicking it fires the matching move.
4. `Board.tsx` no longer renders the legacy bottom button; the
   chief gets `chiefEndPhase` from inside `ChiefPanel`, every other
   role gets the new `<role>SeatDone` button from inside its panel.

## Tests

- `scienceSeatDone` flips `G.othersDone[seat]` and (with all three
  non-chief seats done) advances to `endOfRound`.
- Out-of-stage call returns INVALID_MOVE.
- A wrong-role seat call returns INVALID_MOVE.
- `Board.tsx` no longer renders an "End my turn" button outside
  of role panels (verify by absence in a smoke test).

- Per-stage interrupts (event cards) that should NOT advance the

  one-way; the round-end hook resets it.
