# 14.8 — Favicon

**Parent:** [14-playtest-followups.md](14-playtest-followups.md)

## Goal

Stop the 404 on `/favicon.ico` that fires on every page load — see
playtest issue V9. Cosmetic but it noisily clutters every dev
console.

## Files

- Create `public/favicon.ico` (or `public/favicon.svg`).
- Modify `index.html` — link the favicon.

## Steps

1. Drop a 32×32 PNG (or simpler: a tiny inline SVG) at
   `public/favicon.svg` with a settlement-ish glyph (a circle for
   the mat, four quadrants for the roles; or just a bold "S").
2. `<link rel="icon" type="image/svg+xml" href="/favicon.svg">` in
   `index.html`. Vite copies `public/` straight into `dist/` so
   the production build picks it up.
3. Confirm by reloading and checking the dev console for the 404.

## Tests

- Manual: dev console clean.
- e2e smoke (12.5) doesn't need to change — Playwright already
  asserts the BoardShell renders.

## Dependencies

None.

## Out of scope

- Favicon variants for high-DPI Apple touch icons. Single SVG is
  enough; browsers scale.
