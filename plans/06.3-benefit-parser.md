# 06.3 — Benefit-string parser

**Parent:** [06-domestic.md](06-domestic.md)

## Goal

Translate the human-readable `benefit` strings in
[buildings.json](../src/data/buildings.json) into typed yields the engine
can sum.

Example: `"2 food and 1 production"` → `{ food: 2, production: 1 }`.

## Files

- Create `src/game/roles/domestic/parseBenefit.ts`.
- Create `tests/roles/domestic/parseBenefit.test.ts`.

## API

```ts
import type { ResourceBag } from '../../resources/types';

export interface BenefitYield {
  resources: Partial<ResourceBag>;
  // Effects that aren't pure yields (modify combat, reduce upkeep, ...).
  effects: BenefitEffect[];
}

export type BenefitEffect =
  | { kind: 'attack'; amount: number }
  | { kind: 'defense'; amount: number }
  | { kind: 'happiness'; amount: number }
  | { kind: 'unitMaintenance'; amount: number } // negative = reduces upkeep
  | { kind: 'unitCost'; amount: number };       // negative = cheaper units

export const parseBenefit: (benefit: string) => BenefitYield;
```

## Steps

1. Tokenize on `" and "` and commas. Each token matches:
   - `(\d+)\s+(\w[\w ]*)` for `"2 food"`, `"4 gold"`, etc.
   - special phrases: `"attack +N"`, `"defense +N"`,
     `"+N Happiness"`, `"decrease unit maintenance by N"`,
     `"units cost N less"`.
2. A single canonical lookup table maps the lowercased noun to either a
   `Resource` (then it goes in `resources`) or to a `BenefitEffect.kind`.
3. Unknown tokens throw with the offending string — failure is loud so we
   notice when content adds new verbs.
4. Empty string → `{ resources: {}, effects: [] }`.

## Tests

- `"2 food"` → `{ resources: { food: 2 }, effects: [] }`.
- `"2 food and 1 production"` → both resources.
- `"attack +1"` → `{ effects: [{ kind: 'attack', amount: 1 }] }`.
- `"decrease unit maintenance by 2"` →
  `{ effects: [{ kind: 'unitMaintenance', amount: -2 }] }`.
- `"units cost 1 less"` →
  `{ effects: [{ kind: 'unitCost', amount: -1 }] }`.
- All 17 strings in `buildings.json` parse without throwing — drive this
  test from the loader.
- Unknown verb throws with the verb in the message.
