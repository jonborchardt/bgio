# 06.2 — domesticBuyBuilding & domesticUpgradeBuilding

**Parent:** [06-domestic.md](06-domestic.md)

## Goal

Two moves: buy a building from hand into a grid cell, and upgrade an
in-play building.

## Files

- Create `src/game/roles/domestic/buy.ts`.
- Create `src/game/roles/domestic/upgrade.ts`.
- Register in `src/game/moves.ts`.
- Create `tests/roles/domestic/buy.test.ts`, `tests/roles/domestic/upgrade.test.ts`.

## API

```ts
export const domesticBuyBuilding: Move<SettlementState>;
// args: cardName: string, x: number, y: number

export const domesticUpgradeBuilding: Move<SettlementState>;
// args: x: number, y: number, upgradeCardName: string
```

## Steps for `domesticBuyBuilding`

1. Validate caller seat holds `domestic` and is in stage `domesticTurn`.
2. Validate `cardName` is in `G.domestic.hand`.
3. Look up `BuildingDef` from `BUILDINGS` (01.2). Cost field is `cost: number`
   (gold) — for V1, treat as `{ gold: cost }`.
4. Validate `canAfford(G.wallets[playerID], { gold: cost })`.
5. Validate `isPlacementLegal(grid, x, y)`.
6. `transfer(wallet, bank, { gold: cost })`.
7. Remove card from hand. Insert
   `{ defID: cardName, upgrades: 0, worker: null }` at `cellKey(x,y)`.

## Steps for `domesticUpgradeBuilding`

1. Same caller / stage validation.
2. Validate cell exists. Validate `upgradeCardName` is the building's
   upgrade target (data-driven; until upgrade data exists, V1 treats
   "upgrade" as `building.upgrades += 1` at a delta cost = 50% of the
   original cost — encoded as a stub until proper upgrade data lands).
3. `transfer(wallet, bank, { gold: deltaCost })`.
4. Increment `building.upgrades`.

## Tests

- Buy: removes from hand, places on grid, deducts wallet, credits bank.
- Buy on illegal placement: `INVALID_MOVE`.
- Buy with insufficient funds: `INVALID_MOVE`.
- Upgrade increments counter; deducts delta cost.
- Upgrade on empty cell: `INVALID_MOVE`.

## Dependencies

01.2, 03.4 (wallets), 06.1.

## Out of scope

- Adjacency *bonuses* (06.5) — this just enforces placement adjacency.
- Real upgrade content. Stubbed at delta = 50%.
