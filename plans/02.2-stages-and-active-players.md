# 02.2 — Stages & active players

**Parent:** [02-engine.md](02-engine.md)

## Goal

During `othersPhase`, every non-chief player should be able to act in their
own role's stage simultaneously, and any role should be able to enter a
short-lived "playing event card" interrupt stage without ending their turn.

## Files

- Create `src/game/phases/stages.ts`.
- Modify `src/game/phases/others.ts` to use the stage map.
- Create `tests/stages.test.ts`.

## API

```ts
export const STAGES = {
  scienceTurn: 'scienceTurn',
  domesticTurn: 'domesticTurn',
  foreignTurn: 'foreignTurn',
  playingEvent: 'playingEvent',
  awaitingChiefDecision: 'awaitingChiefDecision', // for trade-request discard
  done: 'done',
} as const;

export type StageName = typeof STAGES[keyof typeof STAGES];

// Build the activePlayers map for othersPhase from current role assignments.
export const activePlayersForOthers = (
  assignments: Record<string, Role[]>,
): { [playerID: string]: StageName };

// Move helpers — wrap bgio events.setStage / events.endStage.
export const enterEventStage = (events: Events, returnTo: StageName) => void;
export const exitEventStage = (events: Events) => void;
```

## Steps

1. `activePlayersForOthers` returns one entry per seat:
   - `chief` seat → `done` (already finished their phase).
   - other seats → the stage matching their primary non-chief role
     (`science`/`domestic`/`foreign`). If a seat holds two roles
     (game-design.md §Players 1–3p), pick the higher-priority one and
     allow swapping via 04.2's helper.
2. `enterEventStage` stashes the previous stage on a stack inside
   `G._stageStack[playerID]` and calls `events.setStage('playingEvent')`.
3. `exitEventStage` pops back; throws if stack underflow.
4. `othersPhase.turn.onBegin` calls
   `ctx.events.setActivePlayers({ value: activePlayersForOthers(...) })`.

## Tests

- 4-player game: after `chiefEndPhase`, every non-chief seat is in its
  expected stage simultaneously.
- 2-player game: seat 1 holds Domestic+Foreign; default stage is `domestic`,
  swap helper moves it to `foreign`.
- `enterEventStage` then `exitEventStage` returns the seat to its prior
  stage; double-exit throws.
