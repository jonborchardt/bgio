# 11.4 — ScienceBot

**Parent:** [11-ai.md](11-ai.md)

## Goal

Greedy heuristic: pour wallet into the cheapest reachable card; complete
when ready.

## Files

- Create `src/game/ai/scienceBot.ts`.
- Create `tests/ai/scienceBot.test.ts`.

## API

```ts
export const scienceBot: { play: (state) => BotAction };
```

## Steps

1. Find the lowest-level non-completed card per color (the rule lets us
   only advance these). Filter to those whose remaining cost fits within
   wallet.
2. If any are completable now (paid + wallet ≥ cost): pour exactly the
   delta and call `scienceComplete`.
3. Otherwise: pour wallet into the *single* card with the smallest
   remaining cost.
4. End with the seat-done flag (whatever 02.2 uses; e.g. a `done` move).

## Tests

- A wallet that completes one card → bot completes it.
- A wallet that doesn't complete any → bot pours into the cheapest.
- Two cards tied: bot picks deterministically (e.g. by id sort).
- Per-round completion limit (1) honored — if already completed once,
  bot skips completion.
